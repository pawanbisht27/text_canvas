Text Canvas App

A simple and interactive Text Canvas application built using Flutter. This app allows users to create and customize text designs on a canvas with basic editing features.

Features
Add and edit text
Change text size and color
Move text freely on canvas
Simple and clean UI
Save design locally
Tech Stack
Flutter (Dart)
Local Storage
Project Structure
lib/
 ├── main.dart
 ├── screens/
 ├── widgets/
 ├── models/
 └── services/
Installations 
Clone the repo
Run flutter pub get
Run flutter run
